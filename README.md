# 诗意茂名地图 - 初赛 Demo

> 为茂名100个地标写一首诗，打造全民参与的"诗歌地理"。
> TRAE AI 创造力大赛参赛作品

## 项目简介

《诗意茂名地图》是一个基于 Web 的交互式文化地图应用。用户在地图上浏览茂名五区两县的100个地标，认领心仪坐标并投稿原创诗歌（限150字），最终形成一张由市民共同书写的"诗意茂名"。

## 核心功能

| 功能 | 说明 |
|------|------|
| 交互式地图 | 基于 MapLibre GL + 高德地图瓦片，中文标注，10大主题分类筛选 |
| 坐标认领系统 | 48小时倒计时认领，每人最多同时认领3个地标，超时自动释放 |
| 诗歌投稿 | 认领后方可投稿，150字限制，实时字数统计 |
| AI 润色 | 投稿后可选 AI 诗歌润色，对比原稿与优化版本 |
| 海报生成 | Canvas 绘制中国风诗歌海报，支持下载 PNG 分享朋友圈 |
| 诗歌馆 | 按主题浏览所有已投稿诗歌，点赞互动 |
| 管理后台 | 审核投稿、查看统计数据 |

## 技术栈

- 纯前端 MVP（HTML + CSS + JavaScript）
- MapLibre GL 地图引擎
- 高德地图中文瓦片
- LocalStorage 本地数据持久化
- HTML5 Canvas 海报生成

## 项目数据

- **地标数量**：100个（覆盖茂南区、电白区、高州市、化州市、信宜市）
- **主题分类**：10大类（山海胜境、物华天宝、古郡遗韵、红色热土、油城新貌、美丽乡村、人间烟火、工业脊梁、杏坛流芳、向海图强）
- **坐标来源**：高德地图 API 批量地理编码

## 创意亮点

1. **文化 + 科技**：用数字地图承载地方文化记忆，让诗歌成为城市的新坐标
2. **全民共创**：100个地标开放认领，每个人都可以为家乡写诗
3. **AI 辅助创作**：降低诗歌创作门槛，AI 润色帮助用户提升表达
4. **社交传播**：诗歌海报一键生成，便于朋友圈分享传播
